## Installations

```bash
$ npm i -g @nestjs/cli
```

```bash
$ yarn install
```

## Runing Database

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
